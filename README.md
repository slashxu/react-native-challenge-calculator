## 🚀 Instalando Projeto

Para instalar o projeto, siga estas etapas:

Clone o repositório

```
https://github.com/slashxu/react-native-challenge-calculator.git
```

Baixe as dependências

```
npm install ou yarn install
```

Rodando o app

```
expo start
```
